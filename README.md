# Decorators

## Authentication vs Authorization

### Authentication (автентифікація) (“Who are you?”)

Authentication () = **підтвердження особи користувача**.

- Логін/пароль
- Token (JWT)
- OAuth (Log in with Google)

Питання: **“Чи ти справді той, за кого себе видаєш?”**

Якщо успішно — система знає:

> Це user_id = 42, name="Alice".
> 

---

### Authorization (авторизація) (“Що тобі можна робити?”)

Authorization = **перевірка прав доступу**.

- Чи може цей користувач читати ресурс?
- Чи може він видаляти, редагувати, створювати?
- Чи має роль `admin`, `manager`, `student`, etc.

Питання: **“Маючи цю особу, чи можна їй виконати цю дію над цим ресурсом?”**
